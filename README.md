## Project Overview
This is a professional, responsive portfolio website showcasing my skills, projects, and contact information. The website demonstrates proficiency in HTML5 and CSS3, including semantic markup, modern layout techniques (Flexbox/Grid), and responsive design principles.

---

## Features

### Sections
1. Home/Hero - Introduction with name, tagline, and call-to-action buttons
2. About - Professional bio (150+ words), profile image, and skills list
3. Skills - Grid of technical skills with hover effects
4. Projects - 4 project cards with title, description, tech tags, and links
5. Contact - Form with validation and contact information

### Technical Highlights
- Semantic HTML5(header, nav, main, section, article, footer)
- CSS Custom Properties for consistent theming
- Flexbox used in header, about section, and footer
- CSS Grid used in hero, skills, projects, and contact sections
- Mobile-first responsive designwith 3 breakpoints
- Accessibility features(skip link, alt text, form labels)
- CSS Animations and transitions
- Dark mode toggle (bonus feature)
- Print stylesheet (bonus feature)
